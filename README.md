# Finza
Dự án quản lý thời gian, công việc hằng ngày và quản lý chi tiêu.

## Cách chạy project
* Run flutter pub get để tải các dependence
  * Run flutter gen-l10n để gen các file đa ngôn ngữ
  * Run flutter pub run build_runner build --delete-conflicting-outputs 
  để tạo ra các file model và các file code cho các asset
  * Run project
